Software Engineering Project Management

Software Engineering Project Management was my fourth module for the University of Essex Online MSc Computer Science course. I started this 12-week module on 12th March 2024.

Click on the link in the 'Description & Link' column to navigate to the respective piece of work.

Unit Work

Name Unit(s) Description & Link
Collaborative Discussion 1 1, 2, 3 Read an article about project failures and answer questions about our opinions on causes of project failure.
Gherkin Statements 2 Create a sequence of Gherkin statements describing some behaviours involved in interacting with a new coffee machine.
Data Structures 3 Consider what data structures might be used in the backend of an online system that you frequently use.
Collaborative Discussion 2 5, 7, 8 Review a model for measuring user experience and consider how we might adapt it for changing emotions over time.
PyTest 6 Run some unit tests with PyTest on a simple wallet. Change the code to make the tests fail.
Components of User Experience Model 7 Discuss how a Project Manager might approach an emotional customer.
Improving Code Quality 9 Follow guidance on improving overall code quality, then Implement improvements on some of my old code.
Comparing Linters 10 Test some bad code against multiple code quality tools and compare the outputs.
